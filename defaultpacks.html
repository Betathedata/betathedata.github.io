Default resource pack collection

This is a collection of the default assets of Minecraft starting from version 15w31a (The first Minecraft snapshot)

How do I make a resource pack 1.9 compatible?

All you have to do is change "pack_format" in your pack.mcmeta file to 2 instead of 1 and then restart your minecraft client.

An example pack.mcmeta file for a 1.9 resource pack can be downloaded here
