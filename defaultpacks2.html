What is this page?

This is a collection of the default assets of Minecraft starting from version 15w31a (The first Minecraft 1.9 snapshot). It's not very well maintained but whenever I remember, I upload the default resource packs to this page, it really has no use, you can find empty resource pack structures aswell which might help a bit. All downloads are hosted by Mediafire so I can easily track stuff.
